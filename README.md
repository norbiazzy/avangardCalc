# Bonolit Premium Calculator

React-проект без TypeScript.

## Запуск
```bash
npm install
npm start
```

Внутри: BLOCKS со штуками в палете, пересчёт м³/шт/палет, localStorage, premium UI, PDF КП, Telegram, Firebase-заготовка.

Telegram: `src/utils/sendTelegram.js` — вставить токен бота и chat_id.
Firebase: `src/utils/firebase.js` — вставить config из Firebase Console.
